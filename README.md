Nexum is a privacy-first financial management app for iOS. It allows you to securely track your expenses, manage budgets, and receive personalized financial adviceâ€”all processed on-device with no data ever leaving your phone.
Features

    Expense Tracking: Log and categorize your transactions with ease.
    Budget Management: Set, track, and adjust budgets to keep your finances in check.
    Personalized Financial Advice: Receive tailored tips based on your spending patterns, powered by an on-device LLM (Local Learning Model).
    Privacy-First: Your data is never transmitted. All processing happens locally, ensuring maximum privacy.
    Security: Sensitive data is encrypted using industry-standard techniques.

Installation

    Clone the repository:

    bash

    git clone https://github.com/ashuklaa/nexum.git
    cd nexum

    Open the project in Xcode:
        Ensure you have the latest version of Xcode installed.
        Open the .xcodeproj file in Xcode.

    Build and run:
        Select your target device or simulator.
        Build and run the app.

Usage

    Tracking Expenses: Use the transaction entry screen to log your daily expenses. Categorize each transaction for better tracking.
    Managing Budgets: Set your monthly budgets for different categories. The app will help you stay within your limits.
    Receiving Advice: Check the personalized tips on the dashboard to find new ways to save based on your spending patterns.
